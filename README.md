Tool to download playlists or specific Youtube videos using YTP
Requires ffmpeg on the device.

USAGE
For downloading playlists
  Change playlist URL on ytp.py to your playlists
  run python ytp.py and press enter on the prompt
  On playlist changes, rerun the two steps above to update
  
For downloading a specific video
  run python ytp.py and paste the link of the video on the prompt

detector.py can be used to detect videos that have become unavailable
